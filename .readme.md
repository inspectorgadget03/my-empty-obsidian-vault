# NOTE:
Windows users must first install python3 to use the autoindex system, then change 
```
<%+tp.user.autoindex(.... 

to

<%+tp.user.autoindex4windows(....
```

Unix users should change 
```
<%+tp.user.autoindex(.... 

to

<%+tp.user.autoindex4unix(....
```
